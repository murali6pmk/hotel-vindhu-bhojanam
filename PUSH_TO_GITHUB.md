Go to /download on the site and download hotel-vindhu-bhojanam.zip

## Step 2: Extract and open terminal in that folder

## Step 3: Run these commands one by one

```bash
# Install dependencies
npm install

# Initialize git
git init
git add .
git commit -m "🍛 Hotel Vindhu Bhojanam - Full restaurant website"

# Push to YOUR GitHub repo
git branch -M main
git remote add origin https://github.com/murali6pmk/hotel-vindhu-bhojanam.git
git push -u origin main
```
